# Rental Manager Android

Android-клиент подключается к существующей FastAPI-панели через JSON API и рисует собственный нативный интерфейс:

- настройка адреса сервера прямо в приложении, чтобы миграция хоста не требовала новой сборки;
- тёмный iOS-inspired интерфейс с крупными заголовками, нижней навигацией и карточками;
- дашборд, жильцы, аренда, ручные оплаты, счётчики, коммуналка, тарифы, расходы, отчёты, сообщения и базовые серверные настройки;
- загрузка Excel/JSON-отчётов через системный DownloadManager;
- фоновые проверки `/api/bootstrap` через JobScheduler;
- детальные настройки уведомлений: частота, тихие часы, звук/вибрация/тихо и отдельные типы событий;
- постоянное уведомление по квартирам-должникам без отсрочки.

Сборка:

```powershell
powershell -ExecutionPolicy Bypass -File .\build-apk.ps1
```

Готовый APK появляется здесь:

```text
android/RentalManager/build/rental-manager-mobile.apk
```

По умолчанию приложение предлагает `http://127.0.0.1:8000`, но на телефоне нужно указать реальный адрес деплоя или IP сервера в сети. Да, localhost телефона - это сам телефон, вот такой маленький сюрприз.
